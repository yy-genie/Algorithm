## 논리와 증명 12번

정수 \(n\)이 3의 배수가 아니면 \(n = 3k+1\) 또는 \(n = 3k+2\)이다.  
이때 \(n^2 = (3k+1)^2 = 3(3k^2 + 2k) + 1\) 또는 \(n^2 = (3k+2)^2 = 3(3k^2 + 4k + 1) + 1\)이므로 \(n^2\)은 3의 배수가 아니다.  
따라서 대우에 의해 \(n^2\)이 3의 배수이면 \(n\)은 3의 배수이다.


## 기초 수식 4번

T(n) = T(n/2) + 1

= T(n/4) + 1 + 1  
= T(n/8) + 1 + 1 + 1  
= ...  
= T(n/2^k) + k

n / 2^k = 1 이 되면 멈추므로  
n = 2^k  
k = log₂ n

따라서

T(n) = T(1) + log₂ n  
= 1 + log₂ n  

O(T(n)) = O(log n)
